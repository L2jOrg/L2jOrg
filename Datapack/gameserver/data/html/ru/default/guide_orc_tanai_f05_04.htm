Умения автоматически появляются в соответствующем окне при достижении уровня, но для их изучения необходимы SP. SP - это очки умений. Их зарабатывают вместе с очками опыта во время охоты. Они нужны для разучивания новых умений. Чтобы просмотреть количество Ваших SP, щелкните окно Информация о персонаже и найдите вкладку Характеристики. Обратите внимание, для изучения разных умений требуется разное количество SP.
